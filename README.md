- A recreation of Honk, an IOS real-time messaging app.
- !NOTE! Compile SASS with DartCSS before running with `python -m app`

# Dependencies/Stack
## Frontend
- HTML/CSS
- Jinja2
- JavaScript
## Backend
- Python:
    - flask
    - flask-session
    - flask-wtf / WTForms
    - Werkzeug
    - flask-socketio / SocketIO
    - Dart SASS 1.97.3
- SQLite
